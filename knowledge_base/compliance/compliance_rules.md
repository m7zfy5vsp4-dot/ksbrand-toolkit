# 金山云合规与审核规范

## 内容合规基本要求

### 禁用词规则

1. 严禁使用《广告法》禁止的绝对化用语
2. 严禁使用未授权的竞品名称或产品名
3. 严禁使用"最"字开头的绝对化表述

### 数据引用合规

1. 所有数据须有RAG知识库来源
2. 不得编造或猜测任何数据
3. 数据须标注统计周期与来源

### 品牌名合规

1. 品牌名统一使用"金山云"
2. 产品名使用官方规范写法（见config.py PRODUCT_NAMES）
3. 禁止使用任何非官方的品牌名缩写

### 免责声明要求

1. AI生成内容须标注"AI辅助生成"声明
2. 营销内容须包含"以官网信息为准"声明
3. 新闻稿须包含授权转载声明

## 审核流程

1. AI生成内容 → 自动合规校验 → 人工终审 → 发布
2. 不合规内容自动拦截并提示修正
3. 人工终审须由品牌合规负责人确认
